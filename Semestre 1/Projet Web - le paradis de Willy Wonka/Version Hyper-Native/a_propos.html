Nous sommes le groupe Disruption. Ce groupe est composé de Victor Guenez-Charles, Thomas Wiscart, Colins Neil Nimalaraj et Annais Tchewo Kewe. Notre but est de rendre l'expérience Web la plus simple possible.

Dans un premier temps, une maquette du site web a été construite en groupe. Elle a servi de base à ce qui allait devenir le nouveau site Web. Le logo du site Web a été conçu par Thomas Wiscart. Le site Web a été dévelopé par le groupe de travail Disruption (voir question précédente).
